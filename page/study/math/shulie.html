01 / 概念引入

什么是数列？

从「一串有规律的数字」说起

想象你在排队，第 1 个人、第 2 个人、第 3 个人……每个位置都有一个对应的编号。数列就是按照一定顺序排列的一列数，每个数都有一个确定的位置。

比如：$1, 4, 9, 16, 25, \ldots$ 这就是一个数列，第 $n$ 项是 $n^2$。又比如：$2, 4, 8, 16, 32, \ldots$ 也是一个数列，第 $n$ 项是 $2^n$。

核心概念：数列中每个数叫项，第 $n$ 个数叫第 $n$ 项，记作 $a_n$。整个数列记作 $\{a_n\}$。如果项数有限，叫有穷数列；如果无限延续，叫无穷数列。
$$\{a_n\}: \quad a_1, \; a_2, \; a_3, \; \cdots, \; a_n, \; \cdots$$

数列的本质是一个定义在正整数集上的函数：$a_n = f(n)$，自变量是位置 $n$，因变量是该位置的值 $a_n$。理解这一点，很多问题就能用函数思维来解决。

数列的两个核心问题：① 通项公式——给定 $n$，求 $a_n$；② 前 $n$ 项和——求 $S_n = a_1 + a_2 + \cdots + a_n$。整个数列的学习围绕这两个问题展开。
02 / 等差数列

等差数列 —— 每次加一样的数

最基本也最常用的数列

如果一个数列从第二项起，每一项与前一项的差都等于同一个常数，就叫等差数列。这个常数叫公差，记作 $d$。

2
3.0
公式 1

通项公式

$$a_n = a_1 + (n-1)d$$

含义：第 $n$ 项 = 首项 + $(n-1)$ 个公差。因为从 $a_1$ 到 $a_n$，一共加了 $(n-1)$ 次 $d$。

公式 2

前 n 项和

$$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n[2a_1 + (n-1)d]}{2}$$

记忆：$\frac{\text{项数} \times (\text{首项} + \text{末项})}{2}$。这就是高斯小时候算 $1+2+\cdots+100$ 用的方法！

高斯的故事：老师让全班算 $1+2+3+\cdots+100$，高斯发现 $1+100=101$，$2+99=101$……共 50 对，所以 $S = 50 \times 101 = 5050$。这就是等差数列求和公式的起源。
等差中项：如果 $a, b, c$ 成等差数列，则 $b = \frac{a+c}{2}$，$b$ 叫 $a$ 和 $c$ 的等差中项。这是判断等差数列的利器。
03 / 等比数列

等比数列 —— 每次乘一样的数

指数增长的数学模型

如果一个数列从第二项起，每一项与前一项的比都等于同一个常数，就叫等比数列。这个常数叫公比，记作 $q$。

1
2.0
公式 3

通项公式

$$a_n = a_1 \cdot q^{n-1}$$

含义：第 $n$ 项 = 首项 $\times$ 公比的 $(n-1)$ 次方。因为从 $a_1$ 到 $a_n$，一共乘了 $(n-1)$ 次 $q$。

公式 4

前 n 项和

$$S_n = \frac{a_1(1 - q^n)}{1 - q} \quad (q \neq 1)$$
1
$S_n = a_1 + a_1 q + a_1 q^2 + \cdots + a_1 q^{n-1}$
2
两边同乘 $q$：$q \cdot S_n = a_1 q + a_1 q^2 + \cdots + a_1 q^n$
3
两式相减：$S_n - qS_n = a_1 - a_1 q^n$
4
整理得：$S_n = \frac{a_1(1-q^n)}{1-q}$
易错提醒：① 公式要求 $q \neq 1$；当 $q=1$ 时，$S_n = n \cdot a_1$。② 求和公式中分子是 $1 - q^n$，不是 $q^n - 1$，注意符号！
等比中项：如果 $a, b, c$ 成等比数列，则 $b^2 = a \cdot c$，$b = \pm\sqrt{ac}$。注意等比中项有两个值（正负）。
04 / 通项求法

如何求通项公式？

从已知条件推导 $a_n$ 的几种方法

方法 1

公式法（等差 / 等比）

如果是等差或等比数列，直接套公式：

$$\text{等差：} a_n = a_1 + (n-1)d \qquad \text{等比：} a_n = a_1 \cdot q^{n-1}$$

关键是求出 $a_1$ 和 $d$（或 $q$），通常用已知条件列方程组。

方法 2

$S_n$ 与 $a_n$ 的关系

已知前 $n$ 项和 $S_n$，求 $a_n$：

$$a_n = \begin{cases} S_1 & (n=1) \\ S_n - S_{n-1} & (n \geq 2) \end{cases}$$
务必注意：$n=1$ 时必须单独验证！$a_1 = S_1$，不能用 $S_1 - S_0$（$S_0$ 无定义）。这是考试中最容易丢分的地方。
方法 3

累加法

形如 $a_{n+1} - a_n = f(n)$ 的递推式，把所有差加起来：

1
$a_2 - a_1 = f(1)$
2
$a_3 - a_2 = f(2)$
3
$a_n - a_{n-1} = f(n-1)$
4
全部相加：$a_n - a_1 = f(1) + f(2) + \cdots + f(n-1)$
5
所以：$a_n = a_1 + \sum_{k=1}^{n-1} f(k)$
方法 4

累乘法

形如 $\frac{a_{n+1}}{a_n} = f(n)$ 的递推式，把所有比乘起来：

$$a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k) = a_1 \cdot f(1) \cdot f(2) \cdots f(n-1)$$

适用于各项之比为已知函数的情况。

06 / 求和技巧

特殊数列的求和方法

不是等差等比也能求和

技巧 1

裂项相消法

把每一项拆成两项之差，求和时中间项全部抵消：

$$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$$
1
例：求 $\frac{1}{1\times2} + \frac{1}{2\times3} + \cdots + \frac{1}{n(n+1)}$
2
裂项：$= \left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \cdots + \left(\frac{1}{n} - \frac{1}{n+1}\right)$
3
相消：$= 1 - \frac{1}{n+1} = \textcolor{orange}{\frac{n}{n+1}}$
常见裂项：① $\frac{1}{n(n+k)} = \frac{1}{k}\left(\frac{1}{n} - \frac{1}{n+k}\right)$；② $\frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$
技巧 2

错位相减法

等差 $\times$ 等比的数列，用错位相减：

1
设 $S_n = a_1 + a_2 q + a_3 q^2 + \cdots + a_n q^{n-1}$
2
两边乘 $q$：$qS_n = a_1 q + a_2 q^2 + \cdots + a_n q^n$
3
两式相减，中间项错位抵消
4
整理即得 $S_n$
技巧 3

分组求和法

把数列拆成几个等差或等比子数列，分别求和再合并：

$$S_n = (1 + 3 + 5 + \cdots) + (2 + 4 + 8 + \cdots)$$

适用于奇数项和偶数项分别有不同规律的数列。

技巧 4

倒序相加法

把和式正着写一遍、倒着写一遍，再相加：

$$S = a_1 + a_2 + \cdots + a_n$$ $$S = a_n + a_{n-1} + \cdots + a_1$$ $$2S = (a_1+a_n) + (a_2+a_{n-1}) + \cdots$$

等差数列求和公式就是用这个方法推导出来的。

07 / 递推

递推数列

由递推关系求通项

很多时候，题目不直接给通项公式，而是给一个递推关系——告诉你 $a_{n+1}$ 和 $a_n$ 之间的关系。下面看看如何从递推中求出通项。

类型 1

$a_{n+1} = a_n + f(n)$ 型（累加）

例：$a_1 = 1$，$a_{n+1} = a_n + \frac{1}{n(n+1)}$，求 $a_n$。

1
$a_n - a_{n-1} = \frac{1}{(n-1)n} = \frac{1}{n-1} - \frac{1}{n}$
2
累加：$a_n = a_1 + \sum_{k=1}^{n-1}\left(\frac{1}{k} - \frac{1}{k+1}\right)$
3
裂项相消：$= 1 + \left(1 - \frac{1}{n}\right) = \textcolor{orange}{2 - \frac{1}{n}}$
类型 2

$a_{n+1} = f(n) \cdot a_n$ 型（累乘）

例：$a_1 = 1$，$a_{n+1} = \frac{n+1}{n} a_n$，求 $a_n$。

1
$\frac{a_{n+1}}{a_n} = \frac{n+1}{n}$
2
累乘：$a_n = a_1 \cdot \prod_{k=1}^{n-1} \frac{k+1}{k} = 1 \cdot \frac{2}{1} \cdot \frac{3}{2} \cdots \frac{n}{n-1}$
3
约分：$= \textcolor{orange}{n}$
类型 3

$a_{n+1} = p \cdot a_n + q$ 型（构造等比）

设 $a_{n+1} + c = p(a_n + c)$，解出 $c = \frac{q}{p-1}$，则 $\{a_n + c\}$ 是等比数列。

1
例：$a_1 = 2$，$a_{n+1} = 2a_n + 1$
2
设 $a_{n+1} + c = 2(a_n + c)$，展开得 $a_{n+1} = 2a_n + c$，所以 $c = 1$
3
$\{a_n + 1\}$ 是等比数列，首项 $a_1 + 1 = 3$，公比 $2$
4
$a_n + 1 = 3 \cdot 2^{n-1}$，所以 $a_n = \textcolor{orange}{3 \cdot 2^{n-1} - 1}$
核心思路：递推求通项的万能策略——通过变形，把递推关系转化为我们熟悉的等差或等比数列，然后用公式求解。

09 / 实际应用

数列有什么用？

从复利到人口增长，数列无处不在

金融：复利计算

本金 $P$，年利率 $r$，$n$ 年后总额 $= P(1+r)^n$。这是等比数列的直接应用。

生物：人口增长

理想条件下种群数量按等比数列增长。细菌分裂、兔子繁殖都是经典模型。

建筑：阶梯设计

每级台阶高度相同（等差），或逐级递增（变差），都是数列在工程中的应用。

生活：储蓄计划

每月存入固定金额，利息复利计算。等差（本金）+ 等比（利息）的组合问题。

艺术：黄金分割

斐波那契数列 $1,1,2,3,5,8,\ldots$ 的相邻项之比趋近黄金比例，广泛出现在自然与艺术中。

计算机：算法复杂度

二分查找 $O(\log n)$、归并排序 $O(n\log n)$，都涉及数列求和与增长分析。
