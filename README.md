ShuoWeb Coming Soon Page
这是一个为 ShuoWeb 量身定制的极简主义“网站建设中”预热页面。该页面完全基于原生技术栈构建，专注于高视觉质量与极致的性能表现。
核心设计理念
 * 视觉美学：采用几何几何美学设计，通过精细的线框与对比度适中的色彩营造专业感。
 * 零媒体依赖：完全不使用外部图片、字体库或 Emoji，所有视觉元素（包括动态图标）均由 行内 SVG 绘制。
 * 双主题系统：集成“浅蓝（默认）”与“浅粉”两套配色方案，支持无缝丝滑切换。
 * 交互动画：利用 CSS3 硬件加速动画实现多维度的悬浮、旋转与呼吸效果，模拟精密仪器的运作感。
技术规格
| 维度 | 技术栈 / 说明 |
|---|---|
| 构架 | 单文件 HTML5 结构 (Single-file solution) |
| 样式 | 原生 CSS3 (使用 CSS Variables 实现主题实时响应) |
| 脚本 | 轻量级 Vanilla JS (处理 DOM 属性切换) |
| 响应式 | 移动端优先 (Mobile-First)，适配所有主流屏幕尺寸 |
| 动画 | 基于 cubic-bezier 的平滑贝塞尔曲线过渡 |
快速开始
 * 下载或复制代码至本地。
 * 将文件重命名为 index.html。
 * 使用任意现代浏览器打开即可查看效果。
自定义配置
您可以直接在 HTML 文件的 :root 部分修改颜色变量，以调整主题色彩：
/* 修改浅蓝主题核心颜色 */
--accent-color: #3B82F6; 

/* 修改浅粉主题核心颜色 */
[data-theme="pink"] {
    --accent-color: #EC4899;
}
