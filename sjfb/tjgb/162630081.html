萧县2024年国民经济和社会发展统计公报[1]

发表时间： 2025-06-20 15:09 信息来源： 萧县统计局 作者： 统计局 初审：王加浩 复审：刘敏 终审： 王亚萍 阅读人次： 字体： [ 大 中 小 ] 保护视力色：

202 4 年 ，在县委县政府的团结带领下，全县坚持以习近平新时代中国特色社会主义思想为指导，全面贯彻落实党的二十大和二十届二中、三中全会精神和习近平总书记关于安徽工作的重要讲话重要指示精神，坚持稳中求进工作总基调，迎难而进、真抓实干，经济持续稳定向好，高质量发展有力推进。

一 、 综合

年末全县户籍人口 136.9 万人 ， 年末常住人口 10 2 . 0 万人 。

初步核算，全年全县生产总值 [2] （ GDP ） 487.62 亿元，同比增长 6.0% 。分产业看，第一产业 [ 3 ] 增加值 76.96 亿元，增长 2.7 % ；第二产业增加值 184.61 亿元，增长 8.7 % ；第三产业增加值 226.05 亿元，增长 4.7 % 。三次产业结构 调整为 15.8 ： 37.9 ： 46.3 。

二、农业

全年农作物播种面积 17.07 万公顷，增长 0.2 % 。粮食作物播种面积 15.15 万公顷，增长 0.1 % 。其中：小麦播种面积 7.51 万公顷，下降 0.1 % ；玉米播种面积 6.48 万公顷，增长 0.1% ；大豆播种面积 0.91 万公顷，减少 0.1% 。油料播种面积 0.32 万公顷，增长 3.3% ；蔬菜播种面积 1.23 万公顷，增长 0.4 % 。

全年粮食总产量 79.18 万吨，增产 0.1 % 。其中：小麦产量 44.47 万吨，增产 0.6% ；玉米产量 32.26 万吨，减产 0.7% ；大豆产量 1.72 万吨，增产 0.5 % 。油料产量 1.16 万吨，增产 4.3% ；蔬菜产量 52.11 万吨，增产 3.4 % ；园林水果产量 44.32 万吨，增产 4.6% 。

年末全县生猪存栏 24.49 万头，减少 19.0 % 。肉类总产量 12.78 万吨，增长 8.5 % ，其中：猪肉产量 7.93 万吨，增长 18.0 % ；牛肉产量 0.3 万吨，增长 2.7% ；羊肉产量 1.72 万吨，减少 12.8% ；禽肉产量 2.83 万吨，增长 1.5 % 。禽蛋产量 9.82 万吨，增长 0.5 % 。奶类产量 0.28 万吨，下降 0.3 % 。水产品产量 0.88 万吨，增长 0.1 % 。

三、工业和建筑业

202 4 年末全县规模以上工业企业 [4] 187 家，比上年 增加 12 家。 规模以上工业增加值增长 9.1 % ，其中，高技术工业增加值 增长 109.6% ，战略性新兴产业产值增长 1.6 % ，农产品加工产值增长 2.6 % 。分经济类型看，国有控股企业增加值 增长 24.7 % ，股份制企业增加值增长 9.1 % ，外商及港澳台商投资企业增加值增长 10.7 % 。

规模以上工业统计的主要产品产量中，小麦粉 下降 67.5 % ，饲料下降 38.5 % ，纱下降 74.5 % ，机制纸及纸板 下降 3.5 % ，塑料制品 增长 66.7 % ，耐火材料增长 1.6 % ，硅酸盐水泥熟料 增长 2.9 % ， 水泥下降 9.7% 。

全县规模以上工业企业主营业务收入比上年下降 1.7 % ；利润总额 增长 8.6 % 。

全年建筑业增加值 73 亿元，增长 13.8 % 。年末具有资质等级的总承包和专业承包建筑业企业 61 家，比上年增加 2 家。

四、固定资产投资

全年固定资产投资 [5] 比上年增长 11. 1 % 。其中，一、二产投资分别增长 182.3 % 、 12.7 % ，三产投资下降 1.2 % 。其中，工业投资 同比 增长 12.7 % ，技术改造投资增长 48.6 % ，民间投资增长 29.7 % ，建安投资增长 30.8 % ，基础设施投资增长 24.1 % 。

全年房地产开发投资 10.8 亿元，下降 65.6 % 。年末商品房销售面积 22.2 万平方米，下降 63.0 % 。房屋施工面积 107.9 万平方米，下降 37.9 % ；房屋竣工面积 34.0 万平方米，增长 85.3 % 。

五、国内贸易

全年社会消费品零售总额 231.71 亿元，比上年增长 6.7 % 。 按销售单位所在地分， 限上 城镇消费品零售额 15.7 亿元，增长 1 1.1 % ； 限上 乡村消费品零售额 1.3 亿元， 增长 106.7 % 。按消费形态分， 限上 餐饮收入 2.0 亿元， 下降 3.1 % ； 限上 商品零售 15.0 亿元，增长 17.8 % 。

六、对外经济和旅游

全年进出口总额 13764 万美元， 增长 21.6 % 。其中，出口 13554 万美元， 增长 22.5% ；进口 210 万美元， 下降 18.9 % 。年末全县有 4A 级旅游景区 2 处。

20 20 -202 4 年全县进出口额

七、交通和邮电

全年交通运输、仓储和邮政业增加值 28.61 亿元，比去年同期 增长 5.3 ％。全年公路客运量 15 万 人，比上年 增长 25.0 % 。旅客周转量 3185 万 人公里，比上年 增长 31.9 % 。公路货物运输量 6077 万 吨，比上年 增加 4.7 % 。货物周转量 112.4 亿吨公里，比上年 增加 3.4 % 。

年末固定电话用户 30392 户，比上年增加 82 78 户；移动电话用户 692591 户，比上年 减少 272308 户；年末互联网宽带接入用户 457580 户，比上年 增加 49014 户。

八、财政、金融

全年 一般公共预算 收入 19.04 亿元， 下降 14.1 % 。 其 中， 税收收入 11. 32 亿元， 下降 4.5 % 。一般公共预算支出 75.76 亿元， 增长 2.0 % 。其中，一般公共服务支出 4.93 亿元， 增长 2.6 % ；社会保障 和 就业支出 18.13 亿元， 增长 3.6 % ；节能环保支出 0.84 亿元， 下降 40.6 % ；教育支出 1 8. 50 亿元，增长 0.1 % ；卫生健康支出 4.24 亿元， 下降 6.6 % ；城乡社区支出 3.19 亿元， 增长 60.2 % ；农林水支出 17. 77 亿元，增长 0.5 % ；交通运输支出 1.19 亿元， 增长 13.1 % ；科学技术支出 0.5 5 亿元，增长 0.2 % 。

20 20 -202 4 年全县 一般公共预算收入 与支出

年末全县金融机构存款余额 （本外币） 达到 684.99 亿元，比上年末增加 63.63 亿元，增长 10.2 % 。其中，住户存款余额 （ 本外币 ） 578.80 亿元，增长 12.0 % 。金融机构贷款余额 （本外币） 644.83 亿元，比上年末增加 71.33 亿元，增长 12.4 % 。

九、教育和科学技术

县有各类学校 542 所，其中：高中 8 所、初中 49 所、中等职业教育学校 3 所、小学 166 所、幼儿园 315 所、特殊教育学校 1 所。全县在校学生 177493 人，其中：中等职业教育学校 12974 人、高中 19377 人、初中 40751 人、小学 78859 人、幼儿园 25070 人、特教生 462 人。现有教职工 15015 人，其中：中等职业教育学校 909 人、高中 1809 人、初中 3662 人、小学 4894 人、幼儿园 3700 人、特殊教育学校 41 人。现有专任教师 12651 人，其中：中等职业教育 787 人、高中 1462 人、初中 2732 人、小学 5135 人、幼儿园 2501 人、特殊教育 34 人。农村义务教育专任教师本科及以上学历比例 71.8% 。

全年 发明专利授权 41 件，有效发明专利拥有量 760 件 ，每万人发明专利拥有量 7.3 件；商标注册 1793 件，商标有效注册量 12865 件。 科技特派员 292 人，农业技术推广人员 238 人，农业科技进步贡献率 67.5% 。

十、文化、卫生和体育

年末全县有剧场 1 个、影剧院 1 个，体育场馆数 3 个，图书馆 1 个，文化馆 1 个，博物馆 1 个，书画院 1 个，乡镇文化站 23 个，公共图书馆总藏量 225 千册。

年末全县共有医疗机构 559 个，其中：综合医院 16 个、中医医院 1 个、乡镇卫生院 23 个、社区卫生服务中心（站） 20 个、村卫生室 275 个。疾病预防控制中心（卫生监督所） 1 个、妇幼保健院（所、站） 1 个。全县医疗卫生机构卫生技术人员 6005 人，其中，执业（助理）医师 2739 人，注册护士 2553 人。

年末全县体育社团组织机构数 29 个，体育俱乐部机构数 8 个，青少年体育俱乐部 6 个，社区体育健身俱乐部 4 个。全县社会体育指导员 1713 人，全年共开展群众体育年活动 12 次。 全 县在市及以上竞技比赛共获得金牌 54 枚，银牌 55 枚，铜牌 62 枚。

十一、 人民生活和社会保障

全年城乡居民人均可支配收入 25883 元，比去年增长 5.8 % 。其中，城镇 居民 人均可支配收入 40644 元，比去年增长 4.7 % ，农村居民 人均 可支配收入 19234 元，增长 6.1 % 。

20 20 -202 4 年全县全体、城镇和农村人均可支配收入

2024 年城乡居民基本养老保险参保人数 76.61 万人，比上年减少 0.59 万人。 城乡居民医疗保险参保人数 100.18 万人，比去年减少 4.3 万人 ； 失业保险参保人数 3.24 万人，比上年增加 0.16 万人。城市居民最低生活保障人数 0.47 万人，比去年减少 0.04 万人；农村居民最低生活保障人数 4.92 万人，比去年减少 0.16 万人。

年末全县提供住宿的 民政 工作机构 39 个 （含福利院） ，提供住宿的 民政 服务 机构床位 4440 张。

十二、环境

农膜回收率 93.5% ，秸秆综合利用率 98.6% ，水利工程供水能力 26785 万立方米，堤防达标率 100% ，水土保持率 97.7% 。 农村生活污水治理管控 率 22.3 % ，畜禽粪污综合利用率 9 5.0 % 。

初步核算，全年全社会综合能源消费量（当量值） 174.72 万吨标准煤，增长 4.5% ，单位 GDP 能耗 0.3543 吨标准煤 / 万元，下降 1.4% 。

[1] 本公报数据为初步统计数。

[2] 地区生产总值及各产业、行业增加值绝对数按现价计算，增长速度按可比价计算。

[3] 国家统计局对三次产业和行业实行相对分离的划分标准，第一产业指农林牧渔业（不含农林牧渔服务业），第二产业指工业（不含开采辅助活动，金属制品、机械和设备修理业）和建筑业，第三产业指除第一产业、第二产业以外的其他行业。

[4] 规模以上工业统计范围为年主营业务收入 2000 万元及以上的企业。
